# DesafioFinal
Prueba práctica para Quark Academy
Instrucciones:
Se trata de un proyecto en windows forms
Primero abrir la carpeta contenedora con visual studio (yo utilicé Visual Studio 2019)
Una vez dentro dirigirse a la carpeta "Desafío final"
Allí entrar a la carpeta "View" y hacer doble click en Form1.Designer
Seleccionar Form1.Designer como punto de partida para correr el programa
Dentro de la carpeta "Desafío final" estarán las carpetas con los modelos y el presentador, además de la vista en Form1.cs
